# neoncrypto-dashboard
Real-time cryptocurrency dashboard with AI analyst, portfolio tracking, and live market analytics using Binance API and WebSockets.
---
##Features:
- Real-time cryptocurrency price tracking
- Market analytics using charts
- Crypto wallet portfolio simulation
- Login and registration system
- AI-powered crypto market assistant
- Modern glassmorphism UI design
- Live updates using WebSockets
---
##Technologies Used:

HTML5  
CSS3 (Glassmorphism UI design)  
JavaScript (Vanilla JS)  
Chart.js (Analytics charts)  
Binance API (Market data)  
WebSockets (Real-time updates)  
OpenRouter API (AI chatbot)
---
## Project Modules:
### 1. Authentication System
Users can register and login. User information is stored in browser local storage.

### 2. Market Dashboard
Displays real-time cryptocurrency prices with live updates.

### 3. Analytics
Shows 24-hour price change of multiple cryptocurrencies using interactive charts.

### 4. Wallet Simulation
Displays portfolio value based on selected assets and calculates profit/loss.

### 5. AI Crypto Analyst
An AI assistant that answers questions about the cryptocurrency market.
---
## APIs Used

### Binance API
Used to fetch real-time cryptocurrency prices.

### OpenRouter API
Used to power the AI crypto analyst chatbot.
